Track B

Snowball 시스템

내부통제 전 영역(ELC, TLC, ITGC)을 하나의 플랫폼에서 통합 관리하는 솔루션입니다. 고객사의 보안 정책, 예산, 인프라에 따라 최적 도입 방식 선택 가능합니다.

Cloud (구독형, SaaS)

  • • 경제성: 초기 투자 부담 최소화
  • • 신속성: 인프라 구축 없이 즉시 운영 가능
  • • 자동 업데이트: 기능 업데이트 및 유지보수 자동 적용

On-Premise (구축형)

  • • 보안성: 내부망 운영으로 보안 정책 준수
  • • 연동성: 기존 ERP/레거시 시스템 직접 연계
  • • 커스터마이징: 회사 상황에 맞춘 맞춤형 구성

시스템 주요 기능 및 화면

● 1. 종합 대시보드

ELC, TLC, ITGC 평가 결과를 통합 조회하고 종합 리포트를 생성합니다. 회사별/통제 영역별 평가 현황을 한눈에 파악할 수 있습니다.

● 2. RCM 관리

위험통제매트릭스(RCM) 데이터를 중앙 관리하며, 엑셀 기반 파일 업로드 및 버전 관리를 지원합니다. 평가 진행과 자동 연계됩니다.

● 3. 설계 평가

통제활동 설명 검토, 현실 반영도 평가 후 증빙 및 결과를 입력하여 설계 효과성을 종합 판정합니다.

● 4. 운영 평가

테스트 절차 자동 생성, 통제 주기에 따른 표본 크기 자동 계산, 표본별 증빙 및 결과 입력 후 전체 결론을 자동 산출합니다.

Snowball 도입의 실질적 이점 (도입 기대 효과)

업무 효율화

🛡️

🎯

리스크 관리 강화

체계적인 결함 추적과 시정 조치 관리를 통해 내부통제 역량을 극대화하고 컴플라이언스 리스크를 최소화합니다.
