Track B

Snowball System

내부통제 자동화 플랫폼

💻

도입 환경과 요구사항에 따라 맞춤 견적을 제공합니다.

기업 규모 및 통제 항목에 따른 유연한 구성을 지원하며, 클라우드와 온프레미스 중 선택 가능합니다.

☁️ Cloud (구독형)

  • 초기 투자 부담 최소화 (경제성)
  • 인프라 구축 없이 즉시 운영 (신속성)
  • 기능 업데이트 자동 적용

🏢 On-Premise (구축형)

  • 내부망 운영으로 보안 정책 준수
  • 기존 ERP/레거시 시스템 직접 연계
  • 회사 상황에 맞춘 커스터마이징

시스템 주요 기능

1. 종합 대시보드

ELC, TLC, ITGC 평가 결과를 통합 조회하고 종합 리포트를 생성합니다.

2. RCM 관리

위험통제매트릭스(RCM) 데이터를 중앙 관리하며, 엑셀 기반 파일 업로드 및 버전 관리를 지원합니다.

3. 설계 평가

통제활동 설명 검토, 현실 반영도 평가 후 증빙 및 결과를 입력하여 설계 효과성을 종합 판정합니다.

4. 운영 평가

테스트 절차 자동 생성, 표본 크기 자동 계산, 표본별 증빙 및 결과 입력 후 전체 결론을 자동 산출합니다.
